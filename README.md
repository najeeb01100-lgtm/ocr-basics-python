# OCR Basics Project

## 📌 Overview

This project demonstrates the fundamentals of **Optical Character Recognition (OCR)** using Python. It shows how to extract text from images using basic preprocessing techniques and OCR libraries.

The goal of this project is to help beginners understand how OCR works and how to apply it to real-world problems such as digitizing documents, reading scanned text, and automating data extraction.

---

## 🚀 Features

* Load and process images for text extraction
* Apply basic image preprocessing (grayscale, thresholding, etc.)
* Extract text using OCR tools
* Display and analyze extracted text

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* OpenCV (for image processing)
* pytesseract (OCR engine wrapper)
* NumPy
* Matplotlib

## ⚙️ Installation

Follow these steps to run the project locally:

1. Clone the repository:

```
git clone <your-repo-link>
cd <your-repo-folder>
```

2. Install required libraries:

```
pip install opencv-python pytesseract numpy matplotlib
```

3. Install Tesseract OCR:

* Download and install Tesseract from its official site
* Add it to your system PATH
